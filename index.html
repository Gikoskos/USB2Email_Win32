USB2Email

Win32 program that sends an e-mail when a USB device is inserted

Overview

With USB2Email you can send an e-mail whenever a certain USB device is inserted on the computer. Insert your USB device to the PC, find it and choose it from the list, configure your server's SMTP settings and the e-mail you want to send and start the service.

Every time the USB device you chose is inserted on the computer, a single e-mail is sent and the service goes to sleep until your USB device is removed. When it's removed, the program starts running again.

USB2Email logs all the times the USB device is detected and writes down to log files a timestamp of when the USB was inserted.

At the moment the installer isn't finished so the user has to delete the registry key created by this application manually, in case he wants to remove this application completely from the PC.

The registry key to be deleted is

HKEY_CURRENT_USER\Software\USB2Email

Detailed instructions about different functionalities in the application, you will find in the Help dialog.
